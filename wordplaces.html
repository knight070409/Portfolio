Word Places

Word Places is a mobile word puzzle game where players solve vocabulary challenges to unlock and decorate various household items. Designed to blend casual puzzle-solving with home customization, the game provides a relaxing and rewarding experience.

As a Unity Developer, I contributed to both gameplay and visual features in collaboration with the art team and clients. Key responsibilities included:

Worked closely with 2D artists to integrate game-ready assets, ensuring visual consistency and smooth implementation.

Developed the logic for unlocking and displaying house decor items after successful word puzzle completion.

Integrated DoTween for fluid UI transitions and animated interactions, enhancing the overall user experience.

Implemented particle systems—such as confetti bursts—to reinforce player achievements and create satisfying visual feedback.

This role allowed me to sharpen my skills in UI animation, visual feedback systems, and collaborative development in a production environment.
