Fragile Flight (under development)

Fragile Flight is a high-speed endless runner developed for a game jam, featuring a drone as the main character. Inspired by the gameplay style of Subway Surfers, the player must guide the drone through challenging environments, collecting coins and avoiding obstacles to prevent crashing. The game focuses on fast reflexes, precision control, and maintaining flight through increasingly difficult terrain.

In its current version, developed under time constraints for the game jam, the game uses placeholder 3D assets and basic mechanics. However, I’m actively working on evolving it into a polished release with optimized systems, original visuals, and more immersive gameplay.

As the Unity Developer, I’ve been responsible for designing and implementing core gameplay mechanics, handling collision and flight control systems, and laying the foundation for a fully realized mobile experience. The project is set to grow from its initial prototype into a feature-rich game with smoother mechanics, seamless transitions, and enhanced visual polish.
