Mission Ganga (Development discontinued)

Mission Ganga is an HTML5-based endless runner game built using the Phaser 3 framework. Inspired by the Government of India's Project MISSION GANGA, which aims to clean the Ganges River, the game challenges players to navigate through environmental obstacles while promoting awareness of the river's conservation efforts.

As a Game Developer on the project, I was responsible for designing and developing a dynamic level, focusing on creating engaging gameplay mechanics and an immersive level design to enhance the player experience. Using HTML, CSS, and JavaScript, I implemented and optimized game features for smooth browser-based gameplay.

This game was developed as part of an academic project by my team, but future development has been discontinued. Nonetheless, the project provided valuable hands-on experience in game development with Phaser 3 and an opportunity to explore a meaningful theme.
